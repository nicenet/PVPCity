Чат

Список команд, которые используются игроками.


Актуальность, полнота и точность информации не гарантируется.

/reg пароль пароль - для регистрации на сервере.
На нашем сервере можно регистрировать несколько учётных записей с разными никнэймами и одновременно играть за них.

/l пароль - для входа в игру.
Чтобы никто не смог воспользоваться вашим никнэймом и ресурсами, если не знает пароль.

/kit - посмотреть список доступных наборов.

/kit start - получить стартовый набор предметов.
Можно использовать каждые 24 часа.

/kit almaz - получить 1 алмаз.
Можно использовать каждый час.

/kit 1000 - получить дополнительные 1000 игровой валюты.
Можно использовать каждые 24 часа.

/sethome - установить основной дом.
Изменить установленный дом можно этой же командой.

/home - телепортироваться в основной дом.

/sethome название - установить дополнительный дом.
Всего, учитывая основной, можно установить 5 домов.

/home название - телепортироваться в дополнительный дом.

/homelist - посмотреть список ваших домов.

/delhome название - удалить выбранный дом.

/spawn - вернуться на спавн.

/warplist - посмотреть список доступных для телепорта варпов.

/warp название - переместиться на указанный варп.

//wand - получить деревянный топор, с помощью которого выделяются точки для привата.

//pos1 и //pos2 - выбрать точки для привата региона. Координатами выбирается тот блок, на котором стоите.

//hpos1 и //hpos2 - выбрать точки для привата региона. Координатами выбирается тот блок, на который смотрите.

//expand - увеличить выделение региона на указанное количество блоков в указанном направлении.

/rg claim название региона - заприватить регион. Тогда игроки, которые не добавлены в приват не смогут ломать блоки, открывать сундуки, двери и тому подобное..

/rg delete название региона - удалить приват. Команду может использовать только владелец привата.

/rg info - посмотреть информацию о привате, в котором находитесь.

/rg info название вашего региона - посмотреть информацию о вашем привате. Команду может использовать только владелец привата.

/rg addowner название региона никнэйм - с этой командой нужно быть осторожными. Добавить игрока в свой приват как владельца. В этом случае, игрок сможет добавлять новых игроков, удалить вас из привата, устанавливать разрешённые для игроков флаги, которые указаны ниже, строить и разрушать, открывать сундуки, взаимодействовать с существами, использовать механизмы и т.п.

/rg addmember название региона никнэйм - менее опасная команда. Добавить игрока в свой приват как пользователя. Он сможет только ломать и строить, открывать сундуки, использовать механизмы и взаимодействовать с существами и т.п.

/rg removeowner название региона никнэйм - удалить владельца из привата. Команду может использовать только владелец привата.

/rg removemember название региона никнэйм - удалить пользователя из привата. Команду может использовать только владелец привата.

/rg list - посмотреть список регионов, где вы владелец.

Флаги.

Все флаги может устанавливать только владелец привата.

allow - включает разрешение, deny - устанавливает запрет, none - убирает установленный флаг, делает стандартным.

/rg flag название региона use allow/deny/none - использование в привате всеми игроками кнопок, рычагов, дверей и т.п. Если не устанавливать, то использовать могут только добавленные в приват игроки.

/rg flag название региона chest-access allow/deny/none - возможность всем игрокам открывать сундуки, раздатчики и т.п. Если не устанавливать, то открывать могут только добавленные в приват игроки.

/rg flag название региона water-flow allow/deny/none - растекание воды в привате. Если не устанавливать, то растекание разрешено.

/rg flag название региона lava-flow allow/deny/none - растекание лавы в привате. Если не устанавливать, то растекание разрешено.

/rg flag название региона enderpearl allow/deny/none - использование жемчуга эндера в привате. Если не устанавливать, то использовать могут, все игроки.

/rg flag название региона vehicle-place allow/deny/none - ставить транспорт в привате. Например, поставить вагонетку. Если не устанавливать, то ставить могут только добавленные в приват игроки.

/rg flag название региона vehicle-destroy allow/deny/none - разрушать транспорт в привате. Например, забрать вагонетку. Если не устанавливать, то разрушать могут только добавленные в приват игроки.

/bal - посмотреть свой баланс игровой валюты.

/pay никнэйм сумма - передать игровую валюту другому игроку.

/afk - сделать уведомление в чат, что вы отошли.

/getpos - посмотреть свои координаты.

/help - посмотреть доступные команды.

/ignore никнэйм - добавить игрока в игнор, чтобы не видеть его сообщения.

/mail send никнэйм сообщение - Чтобы написать сообщение игроку, даже если он не в игре. Когда игрок зайдёт он сможет прочитать письмо.

/mail read - прочитать сообщения, которые были вам отправлены.

/mail clear - удалить все сообщения, которые были вам отправлены.

/w никнэйм сообщение - отправить личное сообщение игроку.

/call никнэйм или /tpa никнэйм - отправить игроку запрос на телепорт к нему.

/tpaccept - принять запрос на телепорт.

/tpdeny - отклонить запрос на телепорт.

/seen никнэйм - узнать, когда игрок заходил в игру. Если игрок в игре, то показывает, сколько времени находится в онлайне.

/time - показывает игровое время.

/changepassword - сменить пароль.

/ma j - зайти на Моб Арену.

/ma l - уйти с Моб Арены.

/tps - проверить производительность сервера. Если все 3 числа зелёные, значит всё отлично.

/back - вернуться на место смерти вашего персонажа.
